    Depending on the version of Minecraft you will play, you need different versions of Java:

    Once you downloaded the Java binary, simply click Next and finish the installation process. You don't need to do anything else.
    You will also need to close and re-open the launcher to see the newly installed java binaries.

  6. If you see a window asking you if you want to fetch the curseforge-api key, click on Yes.

Before starting to play, you need to setup an account to be able to play. To do so:

  1. Click on the Settings button on top of the main launcher menu.
  2. Once the setting tab is opened, click on Accounts and then either click on Add Microsoft if you have a premium account or Add Offline if you did not pay for the game.
Now that you have an account, you can proceed by creating an instance.
  1. To create an instance you need to click on the Add Instance button.
  2. If you want to play modded and you already have the modpack .zip link, you can go in Import zip and enter the link in the field and then click OK.
  3. Now click on Launch instance to Launch the game.
